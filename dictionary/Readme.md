## 单字节fuzz

目的：探测哪些字符（一般是不可见字符）能够在打破规则匹配的同时不影响后端解析。

注意点：在尝试的时候发现0A 0B两个字符在burpsuite发送的时候消失了，可能和协议规范有关。
